|               BIOS ROM                 |
|----------------------------------------| 0xF0000
|        Add-on ROMs (Video BIOS, etc)   |

|               Video RAM                |
|----------------------------------------| 0xA0000
|                                        |
|        Conventional RAM (640KB)        |
|                                        |
|----------------------------------------| 0x00500
|     BIOS Data Area (copied from ROM)   |

RAM assigned for BIOS C code: 0x0500 - 0x10000 (base 64KB)
